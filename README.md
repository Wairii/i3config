# i3config
My i3-gaps config, probably works with regular i3 too

## Requirements
* i3-gaps
* i3status
* feh
* urxvt
* dmenu
* wal - to generate color scheme from wallpaper or other picture
* lxappearance - to change GTK theme
### Fonts
* System San Francisco Display
* Awesome Fonts - for symbols
### GTK
* Theme: Arc-Darker
* Icons: Arc
